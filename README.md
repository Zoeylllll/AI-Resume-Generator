# 🧠 AI Resume Generator | 三语智能简历生成器

> Generate, polish, translate, and export beautiful resumes in Chinese, English, or Korean – powered by OpenAI.

---

## 🌟 项目亮点 | Highlights

- 🧠 一键智能润色简历内容（LLM支持）
- 🌐 中/英/韩 三语界面 + 翻译
- 🎨 支持三种简历模板风格（技术 / 商务 / 学术）
- 📄 支持导出为 PDF / Word (.docx)
- ☁️ 在线部署可体验（Streamlit Cloud）
